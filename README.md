# bilibili

仿b站移动端，使用vue-cli4.x创建项目

路由采用history模式和懒加载的方式

使用vw单位实现自适应

组件库使用的是Vant

图标来自的是阿里巴巴矢量图标库



**后台接口：http://112.74.99.5:3000/web/api**
